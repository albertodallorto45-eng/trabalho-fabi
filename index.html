Sobre o Projeto

Este site foi desenvolvido como parte de uma entrega para a faculdade. O principal objetivo é demonstrar a capacidade de criar interfaces estruturadas, bonitas e que se adaptam a qualquer tamanho de tela (Responsividade).

  • ✓ Estrutura semântica do HTML5
  • ✓ Sistema de Grid do Bootstrap 5
  • ✓ Estilização em CSS Vanilla (Custom)
  • ✓ Mobile-first e responsividade

Tecnologias Utilizadas

Ferramentas aplicadas na construção deste site

HTML5

Estruturação de todas as páginas garantindo semântica, SEO e acessibilidade.

CSS3

Criação de estilos únicos, tipografia moderna e micro-interações para os usuários.

Bootstrap

Uso do framework para acelerar a criação do layout com grid system e componentes responsivos.
